# Lucky Spin - Random Number Wheel

A fun and interactive web application that generates random numbers between 0 and 200 using a spinning wheel animation.

## Features

- Beautiful spinning wheel animation
- Random number generation (0-200)
- Sound effects for spinning and winning
- Responsive design
- User-friendly interface
- Mobile-friendly layout

## How to Play

1. Click the SPIN button in the center of the wheel
2. Wait for the wheel to stop spinning
3. See your lucky number!

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome for icons
- Web Audio API for sound effects

## Project Structure

- `index.html` - Main HTML structure
- `styles.css` - Styling and animations
- `script.js` - Wheel logic and interactions
- `audio/` - Sound effect files

## Setup

1. Clone the repository
2. Open `index.html` in your web browser
3. Start spinning the wheel!
